================================================================================
  Student Records System — Local setup (Windows + XAMPP)
================================================================================
This project has two parts:
  • backend/   — Laravel 12 API (PHP)
  • frontend/  — React app (Create React App)

--------------------------------------------------------------------------------
1. Install XAMPP
--------------------------------------------------------------------------------
  • Download XAMPP for Windows from: https://www.apachefriends.org/
  • Install it (default path is often C:\xampp).
      - PHP 8.2
  After install, add PHP to your PATH (example):
      C:\xampp\php
  Or use full paths like C:\xampp\php\php.exe in the commands below.

  Verify:
      php -v
--------------------------------------------------------------------------------
2. Install Git, Composer, and Node.js
--------------------------------------------------------------------------------
  • Git: https://git-scm.com/download/win
  • Composer (PHP dependency manager): https://getcomposer.org/download/
        Verify: composer -V
  • Node.js LTS (for the React frontend): https://nodejs.org/
        Verify: node -v   and   npm -v

--------------------------------------------------------------------------------
3. Clone the repository and place the project
--------------------------------------------------------------------------------
      git clonehttps://github.com/kevin20040504/CapstoneResearch.git
      cd CapstoneResearch

  For XAMPP, a typical location is:
      C:\xampp\htdocs\CapstoneResearch

  All commands below assume your project root contains `backend` and `frontend`.

--------------------------------------------------------------------------------
4. Backend (Laravel) — install and configure
--------------------------------------------------------------------------------
  Open a terminal in the project root, then:

      cd backend
      composer install

  Create environment file:
      copy .env.example .env

  Generate application key:
      php artisan key:generate

  Set URLs in backend\.env (match how you will run the apps):
      APP_URL=http://127.0.0.1:8000
      FRONTEND_URL=http://localhost:3000

--------------------------------------------------------------------------------
5. Database 
--------------------------------------------------------------------------------

  MySQL (XAMPP)
  -------------------------
  • Start XAMPP Control Panel → start Apache and MySQL.
  • Open phpMyAdmin (http://localhost/phpmyadmin) and create a database, e.g.
        student_records
  • In backend\.env set (adjust name/password as you use locally):
        DB_CONNECTION=mysql
        DB_HOST=127.0.0.1
        DB_PORT=3306
        DB_DATABASE=student_records
        DB_USERNAME=root
        DB_PASSWORD=

  Then:
      php artisan migrate

--------------------------------------------------------------------------------
6. Seed demo accounts (optional but useful for login tests)
--------------------------------------------------------------------------------
  From backend folder:
      php artisan db:seed

  Default test users (change these in production):
      admin    / password123   (role: admin)
      staff    / password123   (role: staff)
      student  / password123   (role: student)

--------------------------------------------------------------------------------
7. Frontend (React) — install and configure
--------------------------------------------------------------------------------
  From project root:

      cd frontend
      npm install

  Create frontend environment file:
      copy .env.example .env

  Default in .env.example:
      REACT_APP_API_URL=http://localhost:8000/api
